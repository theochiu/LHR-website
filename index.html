Installation

Windows/Mac

  1. Download and install snap4arduino
  2. Download and install git
  3. Open git gui and clone this library
  4. Import files and get started!

It's important to note that before starting subsequent times, you must pull from git gui

Operation

Here are some sample programs that can be run inside a snap4arduino environment. The files and sample code for snap4arduino are in /s4a/robot

Incremental Moves

Incremental moves allows you to use the keyboard of the computer to move the liquid handling robot. The left and right arrow keys move the dispenser assembly left and right on the track. The up and down arrow keys move the syringe up and down. The I and K keys move the plunger (mechanism that sucks and releases liquid) up and down.

Preset Cuvette Positions

This program is very usefull since it has shortcuts to the cuvette locations and well locations. Simply line up the bottom rack such that when the syringe is pushed farthest out it lines up over the corresponding hole on the rack. After this calibration the keys 1-9 will take the syringe to the location of the corresponding cuvette while the letters q-u on a querty keyboard should take the syringe to the location of a well. This makes handling liquids easier by eliminating the need to manually move the top assembly using incremental moves.
